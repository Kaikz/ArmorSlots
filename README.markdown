## ArmorSlots

Store blocks/items in your armor slots.

### Requires
- Permissions 2.6+

### Commands
- /hat
- /chestplate
- /leggings
- /boots

### Permissions
- armorslots.hat
- armorslots.chestplate
- armorslots.leggings
- armorslots.boots
